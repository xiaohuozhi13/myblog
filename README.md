# myblog
## 这是一个轻量级的个人版博客项目。
### 项目的设计功能图如下
![Image text](https://www.hualigs.cn/image/6075b736bf475.jpg)
### 项目用到的技术
- 后端：Spring Boot + JPA + thymeleaf模板
- 前端UI：Semantic UI框架
- 数据库：MySQL
### 使用开发工具与环境
- IDEA
- JDK8
- Maven 3
- Axure RP 8
### 项目说明
此项目的定位是一个基于个人使用的博客，故而并不像CSDN、牛客、博客园等社区版支持用户注册登录使用。当然这一点也可以基于本项目去做修改开发，给社区的朋友发挥的空间。
回到项目本身，我用Semantic UI框架实现了比较精美耐看的风格界面图，开发了博客首页界面、分类界面、博客详情、管理员管理博客等页面；
并结合Springboot+JPA+thymeleaf后端技术实现了前端数据与后端数据库的逻辑交互，同时也支持访客对博文的留言评论功能，增加一些交互性。
总体而言，项目已完成最初设计的目标，当然还有很大改进空间，欢迎感兴趣的朋友利用，发挥个人特色，打造属于自己的博客。
